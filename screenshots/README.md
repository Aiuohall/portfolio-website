# Project Screenshots

Drop your project screenshots here, then update each project card in `index.html`.

## Expected filenames (referenced in the HTML comments)
| File | Project |
|------|---------|
| `needynest.png` | File Management System |
| `food-blog.png` | Food Blog Website |
| `supershop.png` | Supershop Management System |
| `opengl.png` | OpenGL / GLUT Graphics Project |
| `barijao.png` | Bari-Jao — Ticket Exchange Platform |

## How to add one
In `index.html`, find the project's `.project-thumb` block and replace the
placeholder with an image, e.g.:

```html
<div class="project-thumb">
  <img src="screenshots/needynest.png" alt="File Management System screenshot">
</div>
```

## Tips
- Use a 16:10 or 16:9 ratio (e.g. 1280×800) for a consistent look.
- Compress them (e.g. [tinypng.com](https://tinypng.com)) so the page stays fast.
- `.png` or `.jpg` both work — `.webp` is even smaller if you can export it.
