# Abu Aiuohall Md Ratul — Portfolio

My personal portfolio website — a Full-Stack & Systems Developer based in Dhaka, Bangladesh (CS @ AIUB).

🔗 **Live:** _coming soon_

## ✨ Features
- Responsive glassmorphic design with dark & light themes
- Animated hero with orbit rings and floating badges
- Project showcase with screenshot and live-demo support
- Working contact form (Web3Forms) with a `mailto` fallback
- SEO optimized — Open Graph tags, structured data, sitemap
- Accessible — reduced-motion support, keyboard focus, skip link

## 🛠️ Built With
- HTML5 & CSS3 (custom properties, no framework)
- Vanilla JavaScript
- [Tabler Icons](https://tabler.io/icons)

## 📂 Structure
| File | Purpose |
|------|---------|
| `index.html` | Main page |
| `styles.css` | All styles |
| `app.js` | Interactions |
| `favicon.svg` | Site icon |
| `robots.txt` / `sitemap.xml` | SEO |
| `404.html` | Custom error page |
| `screenshots/` | Project screenshots (add yours here) |

## 🚀 Run locally
Open `index.html` in a browser, or serve it:

    npx serve .
